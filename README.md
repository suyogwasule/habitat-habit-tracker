Overview

Most habit apps solve one slice of the problem — a streak tracker with no way to actually do the work, a to-do list with no habit framing, a focus timer disconnected from any streak, a journal that never talks to any of it. Habitat is built as one connected loop instead of four separate tools: a trigger to bring you back, a place to do the actual work, a visual reward that proves it's working, and a quick reflection that makes the next reminder worth answering.

Features

Trigger — Reminders
Set a reminder time and a duration for each habit. New habits default to a 21-day window (editable) as a starting finish line, not a rule.

Action — Tasks & Focus Timer
A lightweight to-do list for habits that are a checklist, and a focus timer for habits that are a block of time — whichever fits how the habit actually gets done.

Reward — Completion Calendar
A calendar view of completed days and current streak, one tap away from the habit card, so progress is something you can actually look at.

Investment — Diary
Titled journal entries tied to the day's habit work, turning tomorrow's reminder into a reminder attached to a reason.
